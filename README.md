# AlgoPlayground

AlgoPlayground is an interactive algorithm visualization platform
that helps students understand algorithms step by step using
animations and AI-based explanations.

## Features (Planned)
- Sorting Visualizer
- Pathfinding Visualizer
- Tree Visualizer
- AI-based step explanation
- Web + Mobile support

## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js
- Database: MongoDB
- AI: LLM-based explanations
